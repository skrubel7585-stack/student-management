# Mentora - Mobile Learning Platform

A React Native mobile application that connects students, parents, and mentors through structured lessons and learning sessions.

## Features

- **Multi-role Authentication**: Separate dashboards for Parents, Students, and Mentors
- **Parent Features**: 
  - View and manage students
  - Create new student accounts
  - Monitor student lessons and sessions
- **Student Features**:
  - Personal dashboard with learning stats
  - Browse lessons and sessions
  - View session details
- **Mentor Features**:
  - View assigned students
  - Track student progress
  - Access student lessons and sessions

## Tech Stack

- React Native (Expo SDK 49)
- TypeScript
- React Navigation (Stack Navigator)
- Context API for state management
- Axios (with mock data implementation)

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Expo CLI 
- iOS Simulator (Mac only) or Android Emulator
- Expo Go app on your physical device (optional)
